# PRISMA-App
La App para PRISMA: Proyecto de Reforzamiento e Integración de Saberes MAtemáticos

Esta app se desarrolló para mi servicio social, el cual lo estoy llevando a cabo con Fundación Ceili, y está destinada a servir de apoyo a asesorías de matemáticas para niñas y niños de 10 y 12 años en la comunidad "Tultitlán San Pablo de las Salinas" en el Estado de México.
